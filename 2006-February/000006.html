Rails possui um diretório config no mesmo nível do diretório app, nele ficam
as informações de banco de dados, de routing, ou seja o mapeamento de
actions em urls "mais interessantes" para os usuários e as configurações de
ambiente. Eu acho que seria possível colocar as nossas configurações aqui
também, talvez dentro de uma sub-pasta, mas tem esse problema do usuário
navagenado na estrutura de diretórios de rails, mas este seria um problema
temporário, já que a idéia é que o usuário possa configurar tudo a partir de
uma interface web.
