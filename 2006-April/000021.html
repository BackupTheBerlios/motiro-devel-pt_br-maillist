Sim, dá para fazer isso sem problemas. Assim:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
    "http://www.w3.org/TR/html4/loose.dtd">

<html>
    <head>
        <title>Motiro - JavaScript</title>
        <script type="text/javascript">
            function alteraConteudo(divId) {
                var div = document.getElementById(divId);
                if(div) {
                   div.innerHTML = "<p>Eis o novo texto da div<\/p>";
                }
            }
        </script>
    </head>

    <h4 onclick="javascript:alteraConteudo('div1');">Titulo 1</h4>
    <div id="div1"><p>Texto a ser mostrado 1</p></div>

</html>

Dá para sair adicionando childs na arvore DOM da div tambem, mas acho o
innerHTML mais pratico, ao menos para esse caso.
