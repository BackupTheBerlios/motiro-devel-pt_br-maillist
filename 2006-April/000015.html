Hum, não entendi exatamente, mas deixa eu mostrar um exemplinho que talvez
seja o que vc quer:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
   "http://www.w3.org/TR/html4/loose.dtd">

<html>
    <head>
        <title>Motiro - JavaScript</title>
        <script type="text/javascript">
            function showHide(divId) {
                var div = document.getElementById(divId);
                if(div) {
                    div.style.display = div.style.display == 'none' ?
'block' : 'none';
                }
            }
        </script>
    </head>

    <h4 onclick="javascript:showHide('div1');">Titulo 1</h4>
    <div id="div1" style="display:none;"><p>Texto a ser mostrado 1</p></div>

    <h4 onclick="javascript:showHide('div2');">Titulo 2</h4>
    <div id="div2"><p>Texto a ser mostrado 2</p></div>

</html>

Enfim, a função JavaScript apenas altera o style do objeto. Note que posso
alterar qualquer atributo de um elemento qualquer via JavaScript, já que
tenho total acesso a arvore DOM gerada. Fazer esse tipo de coisa no Firefox
é até bem mais simples, por causa do DOM Inspector. Uma coisa razoavel para
se fazer, é dar ID's para as propriedades mais importante, facilita um
bocado recuperar exatamente aquele elemento. Se não fosse pelo id setado na
div, precisaria fazer alguns passos a mais.
